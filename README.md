# Smart Appointment Booking Platform

A modern, professional appointment booking platform built as a developer portfolio project. Designed with a clean SaaS-style UI, interactive calendar, and smooth user experience. Perfect for showcasing skills in frontend development, form systems, and scheduling logic.

## 🚀 Features

- **Booking Calendar**: Interactive calendar to select available dates.
- **Time Slot Selection**: Choose appointment times easily.
- **Booking Form**: Submit your name, email, phone, selected date, and time slot.
- **Booking Confirmation**: Instant confirmation after scheduling an appointment.
- **Responsive Design**: Fully works on desktop and mobile devices.
- **Modern UI**: Clean typography, subtle gradients, glassmorphism effects, and smooth animations.

## 🛠 Skills Demonstrated

- **Form Systems**: Handling user input and validation.
- **Scheduling Logic**: Managing available time slots and preventing double bookings.
- **Frontend Development**: HTML, CSS, JavaScript for responsive and interactive components.

## 💡 Technologies Used

- HTML5
- CSS3 + Tailwind CSS
- JavaScript (vanilla)
- Responsive design principles
- Glassmorphism and gradient backgrounds
